# Nike Shoes Website

A modern, responsive e-commerce landing page for Nike shoes, built with HTML and CSS. This project features a premium design with smooth animations, interactive elements, and optimized online image sources.

## Features

- **Modern UI/UX**: Premium aesthetic with glassmorphism effects and clean typography.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Interactive Elements**: Hover effects on buttons, navigation links, and product cards.
- **Online Image Integration**: High-quality product images sourced from Unsplash and PngImg.
- **Animated Home Section**: Dynamic entrance animations for a professional feel.

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For custom styling, animations, and responsive layouts.
- **Google Fonts**: 'Poppins' for clean and modern typography.
- **Font Awesome**: Scaling vector icons.

## How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Explore the different sections: Home, Products, Featured, and Reviews.

## Project Structure

- `index.html`: The main entry point containing the website structure.
- `style.css`: Custom stylesheet containing all the premium design rules and responsive media queries.
